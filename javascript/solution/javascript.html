Would you like to play a game?

I have two different versions of the same game for you to make.
Both are games where you will input how many fingers you are holding up, and the computer should randomly guess until it guesses the right number.
Using a named function to run the game means that you can re-use the logic.

In the first game, I have made a drop down menu for you.
This means the user can only select options from the given range.

In the second game, you have a simple input field.
This means you will have to validate the input before running the game.
